The following fonts are recommended for proper display of this document: For body text, the text has been designed around Junicode, a font inspired by the early eighteenth‐century Oxford University Press, and takes advantage of various advanced typographic features if it is installed. For Sevensi text, Unifont LANGDEV should provide basic rendering support in lieu of a more sophisticated solution.

These fonts are not downloaded onto your computer for you (via CSS @font-face or similar), so you will have to install them yourself if you wish to take advantage of their features.

This text is a work in progress, and will be updated with new content as time permits. For a list of changes, see the Changelog.

This document contains MathML markup. For optimal rendering and display, please use a modern MathML‐compatible browser (Safari or Firefox). TeX Gyre Pagella Math is the preferred font for maths rendering, but any maths font installed on your computer should work.
